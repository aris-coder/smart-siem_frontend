# 🖥️ SMART SIEM - FRONTEND

Bienvenue dans le **Frontend** du projet Smart SIEM. Ce service est la vitrine du système. Il transforme les données brutes du backend en dashboards intuitifs, timelines interactives et rapports exploitables.

---

## 🎯 Ce que fait ce frontend

| Fonctionnalité | Description |
| :--- | :--- |
| **Vue Analyste** | Flux d'alertes en temps réel, détails des logs, vue technique des incidents. |
| **Vue RSSI** | KPIs synthétiques, top 5 alertes, carte des sources d'attaque, indicateurs rouge/vert/orange. |
| **Vue Crisis Room** | Plein écran, rafraîchissement automatique (5s), affichage "salle de guerre" pour les décideurs. |
| **Investigation (Timeline)** | Chronologie interactive des événements, pivot en 1 clic sur une IP/Utilisateur. |
| **Reporting** | Génération et visualisation des rapports PDF/Excel. |
| **Gestion des utilisateurs** | Connexion avec MFA, pages de profil, affichage du score UEBA personnel. |
